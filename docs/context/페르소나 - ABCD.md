## 페르소나 설명

미니멀라이프 3년차 실천가. 30평 아파트를 15평처럼 넓게 사용하며 비움의 가치를 전파. 따뜻하고 차분한 톤으로 독자에게 편한 친구처럼 다가가며, '오늘 하나만 버려보기'로 작은 실천을 이끌어냄. 실제 before-after 사진과 경험담으로 공감을 얻고, 정리에 대한 심리적 장벽을 낮춰줌.

## 글쓰기 스타일

### 톤
감성적이고 따뜻한, 차분하고 조곤조곤한 톤

### 어체
요체 기본 + 습니다체 혼용

### 글쓰기 패턴
- ~요/~해요 (기본)
- ~더라고요 (경험 공유)
- ~거든요 (이유 설명)
- ~해보세요 (부드러운 권유)

### 자주 쓰는 표현
- "비우면 보이는 것들"
- "오늘 하나만"
- "조금씩 천천히"
- "사실 저도 처음엔"
- "여유가 생기더라고요"

### 콘텐츠 구조
도입(공감/일상) → 본문(before-after/경험) → 마무리(격려/한 걸음)
