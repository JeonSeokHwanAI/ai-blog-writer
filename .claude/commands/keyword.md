# 키워드 수집 및 분석

인자: $ARGUMENTS

블로그 기반 키워드 분석을 수행합니다. Claude가 WebSearch를 활용하여 분석합니다.

## 사용법 분기

**인자 분석:**
- 인자 없음 → 키워드 입력 요청
- `<키워드>` → 해당 키워드 분석
- `<키워드1>, <키워드2>` → 여러 키워드 분석

## 실행 방법

사용자가 `/keyword <키워드>`를 입력하면 Claude가 다음을 수행합니다:

### 1. 네이버 블로그 검색 분석

WebSearch 도구로 `site:blog.naver.com {키워드}` 검색하여:
- 블로그 글 트렌드 파악
- 주요 콘텐츠 유형 분석
- 인기 블로거 확인

### 2. 연관 키워드 수집

WebSearch로 `{키워드} 관련 검색어` 또는 `{키워드} 추천 키워드` 검색하여:
- 자주 함께 검색되는 키워드 파악
- 롱테일 키워드 발굴

### 3. 검색 의도 분석

검색 결과 제목을 분석하여 사용자 의도 파악:
- 정보/프로필: 누구, 프로필, 경력
- 후기/리뷰: 후기, 리뷰, 솔직, 체험
- 방법/가이드: 방법, 하는법, 만들기
- 추천/비교: 추천, 비교, 순위, 베스트
- 가격/비용: 가격, 비용, 얼마

### 4. 결과 출력

마크다운 표 형식으로 분석 결과 제공:

```markdown
## 🔍 키워드 분석: {키워드}

### 📝 연관 키워드
| # | 키워드 | 유형 |
|---|--------|------|
| 1 | {키워드} 하는법 | 방법 |
| 2 | {키워드} 추천 | 추천 |

### 🎯 검색 의도
| 의도 | 비중 | 추천 콘텐츠 |
|------|------|-------------|
| 방법/가이드 | 높음 | 초보자 가이드 |
| 후기/리뷰 | 보통 | 실제 사용 후기 |

### 💡 콘텐츠 추천
- 가장 많이 검색되는 의도: {의도}
- 추천 제목: "{키워드} 완벽 가이드 (초보자용)"
```

## 출력 예시

```markdown
## 🔍 키워드 분석: AI 바이브 코딩

### 📝 연관 키워드 (발굴)
| # | 키워드 | 검색 의도 |
|---|--------|-----------|
| 1 | AI 바이브 코딩 하는법 | 방법 |
| 2 | 바이브 코딩 뜻 | 정보 |
| 3 | cursor 바이브 코딩 | 도구 |
| 4 | 비전공자 바이브 코딩 | 대상별 |

### 🎯 검색 의도 분석
| 의도 | 비중 | 콘텐츠 형태 |
|------|------|-------------|

### 🏆 블로그 콘텐츠 추천

### 📊 키워드 특성
```

## 골든키워드 판정 기준

### 핵심 공식: 경쟁강도

```
경쟁강도 = 문서수 / 월간검색량

- 경쟁강도 < 1 → 검색 > 글 (골든키워드)
- 경쟁강도 > 1 → 글 > 검색 (레드오션)
```

### 판정 기준표

| 경쟁강도 | 평가 | 표시 | 의미 |
|----------|------|------|------|
| < 0.5 | 골든 | ⭐⭐⭐ | 검색량이 문서수의 2배 이상 |
| < 1.0 | 좋음 | ⭐⭐ | 검색량 > 문서수 |
| 1.0~3.0 | 진입가능 | ⭐ | 경쟁 보통 |
| > 3.0 | 레드오션 | 🔴 | 경쟁 치열 |
| 검색량 < 500 | 수요없음 | ❌ | 검색량 자체가 너무 적음 |

### 결과 표시 예시

```markdown
### 📊 골든키워드 판정

| 키워드 | 검색량 | 문서수 | 경쟁강도 | 판정 |
|--------|--------|--------|----------|------|
| AI 에이전트 | 5,000 | 1,200 | 0.24 | ⭐⭐⭐ 골든 |
| ChatGPT | 100,000 | 500,000 | 5.0 | 🔴 레드오션 |
| 바이브코딩 | 2,000 | 300 | 0.15 | ⭐⭐⭐ 골든 |
| 퀀텀컴퓨팅 | 100 | 800 | - | ❌ 수요없음 |
```

## 주의사항

- Claude가 실시간 웹 검색을 수행하여 분석
- 네이버 블로그 검색 결과 기반으로 트렌드 파악
- 정확한 문서 수는 제공되지 않을 수 있음
- 검색량은 네이버 데이터랩 API 또는 WebSearch로 간접 추정
