# 블로그 리뷰 저장

인자: $ARGUMENTS

## 기능 설명

대화 중 진행한 블로그 글 분석, SEO 개선안, 추천 콘텐츠를 마크다운 파일로 저장합니다.

## 사용법

- `/review-save` → 현재 대화에서 분석한 글 저장
- `/review-save {blog_id} {index}` → 특정 포스트 지정 저장
- `/review-save list` → 저장된 리뷰 목록 보기

## 저장 위치

`docs/reviews/{blog_id}_{제목_sanitized}.md`

## 파일 구조

```markdown
---
# 리뷰 메타데이터
blog_id: "{blog_id}"
post_id: "{post_id}"
url: "{post_url}"
title: "{title}"
author: "{blog_name}"
pub_date: "{발행일}"
reviewed_at: "{리뷰일}"
score: {종합점수}

# 분석 요약
category: "{주제 카테고리}"
strengths: ["{강점1}", "{강점2}", "{강점3}"]
weaknesses: ["{약점1}", "{약점2}", "{약점3}"]
keywords: ["{키워드1}", "{키워드2}", "{키워드3}"]
---

# {title} - 블로그 리뷰

> 원본: {post_url}
> 작성자: {blog_name}
> 발행일: {pub_date}
> 리뷰일: {reviewed_at}

---

## 1. 콘텐츠 구조 분석

(구성요소|내용|평가 표 + 별점)

## 2. 강점 분석

(4가지 강점 상세 설명)

## 3. 약점 및 개선점

(약점|현재상태|개선방안 표)

## 4. SEO 분석

### 현재 제목 분석
(현재 제목의 문제점)

### 추천 제목
(SEO 최적화 제목 5개 + 선정 이유)

### 핵심 키워드
(검색 유입 키워드 5개)

### 추천 메타설명
(50자 이상 설명)

## 5. 독자 반응 예측

(반응유형|예상반응|근거 표)

## 6. 종합 평가

(평가항목|점수/100|코멘트 표)

## 7. 총평

(3-4문장 요약)

---

## 8. 개선 제안

### 추가 섹션 제안
(정보성 콘텐츠 추가안 - 블로그 스타일로)

### 추천 댓글
(작성자에게 남길 댓글)

---

## 나의 메모

{추가 메모}
```

## 동작 흐름

1. 현재 대화에서 분석한 블로그 정보 수집
2. 분석 내용 (7개 섹션) 정리
3. SEO 제목 추천 포함
4. 개선 제안 (추가 섹션, 댓글) 포함
5. 마크다운 파일 생성
6. 저장 완료 메시지 출력
7. **전체분석인 경우**: 블로거 정보 업데이트 제안

## 블로거 정보 업데이트 제안 (전체분석 시)

전체분석(`{blog_id}_전체분석_*.md`) 저장 후, 분석 결과와 `config/blogs.json`을 비교하여 업데이트가 필요하면 자동 제안합니다.

**비교 항목:**
- 설명 (description): 분석의 주제 분포 기반
- 주제(메인): 분석의 메인 주제
- 주제(서브): 분석의 세부 주제들

**출력 형식:**
```
## 리뷰 저장 완료
...

---
### 블로거 정보 업데이트 제안

| 항목 | 현재 | 분석 결과 (제안) |
|------|------|------------------|
| 설명 | AI 활용, 바이브 코딩... | AI 바이브코딩, 비전공자 자동화... |
| 주제(서브) | IT·컴퓨터 | IT·컴퓨터, 교육·학문 |

💡 `/bloggers update {blog_id}` - 업데이트 적용
```

**제안 조건:**
- 전체분석 저장 시에만 제안
- 현재 정보와 분석 결과가 다를 때만 표시
- 변경이 없으면 제안 생략

## list 명령

`docs/reviews/` 폴더의 모든 md 파일 스캔:

```
## 저장된 리뷰 (N개)

| # | 블로그 | 제목 | 점수 | 리뷰일 |
|---|--------|------|------|--------|
| 1 | hhey0510 | 정리수납 봉사... | 77 | 2026-02-11 |
```

## 출력 형식

**저장 완료 시:**
```
## 리뷰 저장 완료

| 항목 | 내용 |
|------|------|
| 파일 | docs/reviews/{filename}.md |
| 블로그 | {blog_id} ({blog_name}) |
| 제목 | {title} |
| 점수 | {score}/100 |

---
`/review-save list` - 전체 목록 보기
```
